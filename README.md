# CultureAI User Guides

This repository contains user-focused documentation for the CultureAI Dashboard. This is separate from the developer documentation to keep user guides and technical docs clearly separated.

## Structure

- `docs/getting-started.mdx` - Introduction and navigation overview
- `docs/copilot.mdx` - Copilot AI assistant guide
- `docs/brand-hub.mdx` - Brand Hub feature guide
- `docs/competitive.mdx` - Competitive analysis guide
- `docs/industry-lens.mdx` - Industry Lens guide
- `docs/content-studio.mdx` - Content Studio guide
- `docs/reports.mdx` - Reports guide
- `docs/quick-reference.mdx` - Quick reference and tips

## Local Development

Preview documentation locally:

```bash
mintlify dev
```

## Deployment

This repository is configured for Mintlify deployment. Connect it to a Mintlify project to deploy.
